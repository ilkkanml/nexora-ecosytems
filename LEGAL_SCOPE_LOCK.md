# Legal Scope Lock

## Product Boundary

Nexora is a legal media player ecosystem.

It is not an IPTV provider.

It is not a broadcaster.

It is not a media/content service.

It is not a channel seller.

It is not a stream relay, stream host, CDN, transcoder, provider marketplace, public catalog, playlist provider, or content distribution platform.

## Allowed Scope

The ecosystem may provide:

- media player applications
- user accounts
- authentication
- role-based access
- customer subscriptions
- software/platform licenses
- device activation
- app-generated device identity
- app version checks
- remote configuration
- maintenance mode
- feature flags
- admin dashboard
- reseller dashboard
- customer dashboard
- audit logs
- payment records for platform access only
- optional temporary encrypted user-owned profile transfer

## Forbidden Scope

The ecosystem must never provide or manage:

- stream hosting
- stream relay
- stream transcoding
- CDN delivery
- channel packages
- channel sales
- IPTV subscription sales
- provider inventory
- content catalog
- public playlist library
- shared playlist marketplace
- permanent backend source of truth for user provider credentials
- copyrighted media files
- pirated channel lists
- unauthorized stream URLs
- hidden bundled sources

## User Responsibility

Users are responsible for entering only lawful provider/profile information they own, license, or are authorized to use.

Nexora does not verify, endorse, provide, sell, distribute, or guarantee third-party content or provider access.

## Public Messaging Rule

Preferred language:

- Legal Media Player
- User-provided playlist player
- Bring your own authorized source
- Player ecosystem
- Device and license management

Avoid:

- IPTV service
- Channel package
- Premium channels
- Free live TV
- Provider access
- Stream service
- Content marketplace

## Final Legal Gate

Every future feature must pass this question:

> Does this keep Nexora as a player/access-management ecosystem, or does it turn Nexora into a content/provider/broadcast service?

If it turns Nexora into a provider, the feature is rejected.
